**Query hooks** — add to stacks/queries.ts:

1. `useStackWires(name: string)` — GET /stacks/{name}/wires
   - Returns { wires: Wire[], unresolved: UnresolvedContract[] }
   - queryKey: ['stacks', name, 'wires']
   - refetchInterval: 30000

2. `useResolveWires(name: string)` — POST /stacks/{name}/wires/resolve
   - useMutation, invalidates ['stacks', name, 'wires'] on success
   - Returns { resolved: number, warnings: string[] }
   - Toast on success: "Resolved {N} wires" or "No changes"

3. `useCreateWire(name: string)` — POST /stacks/{name}/wires
   - useMutation with {consumer_instance_id, provider_instance_id, import_contract_name} body
   - Invalidates ['stacks', name, 'wires']
   - Toast: "Wire created"

4. `useDeleteWire(name: string)` — DELETE /stacks/{name}/wires/{wireId}
   - useMutation, invalidates ['stacks', name, 'wires']
   - Toast: "Wire disconnected"

5. `useCleanupOrphanedWires(name: string)` — POST /stacks/{name}/wires/cleanup
   - useMutation, invalidates ['stacks', name, 'wires']
   - Toast: "Cleaned up {N} orphaned wires"
