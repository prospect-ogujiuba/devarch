**features/instances/queries.ts:**
Add TanStack Query hooks for resource limits:
- `useResourceLimits(stackName: string, instanceId: string)` — GET `/api/v1/stacks/${stackName}/instances/${instanceId}/resources`. Returns `ResourceLimits` or null if 404. Use `queryKey: ['stacks', stackName, 'instances', instanceId, 'resources']`.
- `useUpdateResourceLimits(stackName: string, instanceId: string)` — PUT mutation. Invalidates the resource limits query key on success. Also invalidate the instance detail and effective config queries so those views refresh.
