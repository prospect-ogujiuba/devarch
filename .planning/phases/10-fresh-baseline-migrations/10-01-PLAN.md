    - `services` table: id SERIAL PK, name VARCHAR(128) UNIQUE NOT NULL, category_id INTEGER REFERENCES categories(id), image_name VARCHAR(256) NOT NULL, image_tag VARCHAR(64) DEFAULT 'latest', restart_policy VARCHAR(32) DEFAULT 'unless-stopped', command TEXT, user_spec VARCHAR(64), enabled BOOLEAN DEFAULT true, config_status VARCHAR(20) DEFAULT 'imported' CHECK (config_status IN ('imported','validated','modified','broken')), last_validated_at TIMESTAMPTZ, validation_errors JSONB, compose_overrides JSONB DEFAULT '{}', container_name_template TEXT, created_at TIMESTAMPTZ DEFAULT NOW(), updated_at TIMESTAMPTZ DEFAULT NOW()

    **new_002_service_runtime_shape.up.sql:**
    All tables reference services(id) ON DELETE CASCADE. Create in this order:
    - `service_ports`: id SERIAL PK, service_id INT REFS services(id) ON DELETE CASCADE, host_ip VARCHAR(45) DEFAULT '127.0.0.1', host_port INT NOT NULL, container_port INT NOT NULL, protocol VARCHAR(8) DEFAULT 'tcp', UNIQUE(host_ip, host_port)
    - `service_volumes`: id SERIAL PK, service_id INT REFS, volume_type VARCHAR(16) NOT NULL, source VARCHAR(512) NOT NULL, target VARCHAR(512) NOT NULL, read_only BOOLEAN DEFAULT false, is_external BOOLEAN NOT NULL DEFAULT false
    - `service_env_vars`: id SERIAL PK, service_id INT REFS, key VARCHAR(256) NOT NULL, value TEXT, is_secret BOOLEAN DEFAULT false, encrypted_value TEXT, encryption_version INTEGER NOT NULL DEFAULT 0, UNIQUE(service_id, key)
    - `service_env_files` (NEW): id SERIAL PK, service_id INT NOT NULL REFS, path TEXT NOT NULL, sort_order INT NOT NULL DEFAULT 0, UNIQUE(service_id, path)
    - `service_dependencies`: id SERIAL PK, service_id INT REFS, depends_on_service_id INT REFERENCES services(id), condition VARCHAR(32) DEFAULT 'service_started', UNIQUE(service_id, depends_on_service_id)
    - `service_healthchecks`: id SERIAL PK, service_id INT REFS ON DELETE CASCADE UNIQUE, test TEXT NOT NULL, interval_seconds INT DEFAULT 30, timeout_seconds INT DEFAULT 10, retries INT DEFAULT 3, start_period_seconds INT DEFAULT 0
    - `service_labels`: id SERIAL PK, service_id INT REFS, key VARCHAR(256) NOT NULL, value TEXT, UNIQUE(service_id, key)
    - `service_domains`: id SERIAL PK, service_id INT REFS, domain VARCHAR(256) UNIQUE NOT NULL, proxy_port INT
    - `service_networks` (NEW): id SERIAL PK, service_id INT NOT NULL REFS, network_name TEXT NOT NULL, UNIQUE(service_id, network_name)
    - Add inline indexes on service_id for each table: idx_service_ports_service, idx_service_volumes_service, idx_service_env_vars_service, idx_service_dependencies_service, idx_service_dependencies_depends_on, idx_service_labels_service, idx_service_domains_service

    **new_002_service_runtime_shape.down.sql:**
    - DROP in reverse: service_networks, service_domains, service_labels, service_healthchecks, service_dependencies, service_env_files, service_env_vars, service_volumes, service_ports

    - `service_config_files`: id SERIAL PK, service_id INT NOT NULL REFS services(id) ON DELETE CASCADE, file_path VARCHAR(512) NOT NULL, content TEXT NOT NULL DEFAULT '', file_mode VARCHAR(10) NOT NULL DEFAULT '0644', is_template BOOLEAN NOT NULL DEFAULT false, created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(), updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW(), UNIQUE(service_id, file_path)
    - `service_config_mounts` (NEW): id SERIAL PK, service_id INT NOT NULL REFS services(id) ON DELETE CASCADE, config_file_id INT REFS service_config_files(id), source_path TEXT NOT NULL, target_path TEXT NOT NULL, readonly BOOLEAN NOT NULL DEFAULT false, UNIQUE(service_id, target_path)
    - config_file_id is nullable FK — handles shared/mismatched config cases
    - `service_config_versions`: id SERIAL PK, service_id INT NOT NULL REFS services(id) ON DELETE CASCADE, version INT NOT NULL DEFAULT 1, config_snapshot JSONB NOT NULL, change_summary TEXT, created_at TIMESTAMPTZ DEFAULT NOW()
    - Indexes: idx_service_config_files_service_id, idx_config_versions_service ON (service_id, version DESC)

    **new_003_service_config_model.down.sql:**
    - DROP service_config_versions, service_config_mounts, service_config_files (mounts before files due to FK)
