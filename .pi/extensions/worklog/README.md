# DevArch V2 Worklog Extension

Small project-local pi extension for appending structured DevArch V2 worklog entries.

## Features

- `worklog_append` custom tool for timestamped packet notes
- `/worklog` command to show the active worklog path

## Default log file

`.pi/worklog/devarch-v2.md`

The tool will create the file and parent directory on first use.
